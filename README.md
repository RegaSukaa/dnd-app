# D&D Лист персонажа (Telegram Mini App)

## Установка
1. Склонируйте репозиторий.
2. Загрузите на GitHub Pages или любой другой хостинг.
3. В BotFather создайте бота, установите Menu Button со ссылкой на ваш домен.

## Использование
- Вкладка «Создание» – пошаговый мастер создания персонажа.
- Вкладка «Лист» – просмотр и бросок кубиков.
- Вкладка «Заклинания» – список известных заклинаний.

## Структура
- `js/data/` – игровые данные (необходимо заполнить полностью из оригинальных JSON, предоставленных в анализе).
- `js/engine/` – логика персонажа и правил.
- `js/ui/` – интерфейс.
